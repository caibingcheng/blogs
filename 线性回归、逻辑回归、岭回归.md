# 线性回归、逻辑回归、岭回归

对线性表达式：$A \cdot X + B = Y$.

其中$A, X ,B , Y$都表示矩阵，

我们能够获取已知输入数据$X$，和输入$X$经过系统$G$后的观测数据Y。

## 线性回归：

由于系统误差、观测误差或其他误差，导致观测数据$Y$和输入数据$X$的理论输出$\hat{Y}$ 有误差。

如何修正这个误差？线性回归模型提出了他的方法：

设计误差函数：$\epsilon = \sum(Y - \hat{Y})^2$

展开得：$\epsilon = \sum[Y - (A \cdot X + B)]^2$
